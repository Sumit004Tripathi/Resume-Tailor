# Resume Tailor

Resume Tailor is a modern, client-side React application designed to help users create professional resumes effortlessly. Built with Vite and Tailwind CSS, it features a real-time preview, multiple customizable templates, and instant PDF export capabilities. Users can easily input their details, switch between designs, and download a polished resume in secondsâ€”all directly from their browser.

## Features

*   **Real-Time Preview:** See changes instantly as you type.
*   **Multiple Templates:** Choose from a variety of professional designs (Template A, B, C).
*   **PDF Export:** Download your resume as a high-quality PDF with a single click.
*   **User-Friendly Interface:** Clean and responsive form for entering personal details, experience, education, and skills.
*   **Client-Side Only:** Privacy-focused; all data remains in your browser.

## Tech Stack

*   React
*   Vite
*   Tailwind CSS
*   jsPDF & html2canvas

## Getting Started

1.  Clone the repository.
2.  Run `npm install` to install dependencies.
3.  Run `npm run dev` to start the development server.
